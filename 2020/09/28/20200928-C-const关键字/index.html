C++中的Const关键字

发表于 2020-09-28 更新于 2020-12-11 分类于 C++学习
自己每次看到const都会头疼,然而其实并没有什么大不了的,所以先在这里记录一下

这里的具体内容主要参考C++ Primer Plus 7.3.5小节

const有两种方法用于指针.

第一种方式是让指针指向一个常量const对象,可以防止使用该指针来修改所指向的值.

第二种方法是将指针本身声明为常量,可以防止改变指针指向的位置.
