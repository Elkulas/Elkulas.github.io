记谱法:速度与强弱

发表于 2021-01-07 分类于 音乐
记谱法中关于速度和强弱的标注

速度

基本速度标记

  • Prestississimo - 极快的
  • Prestissimo - 最急板（约为200 - 208 bpm）
  • Prestissimetto － 风之速
  • Allegrissimo － 快而活跃的
  • Presto － 急板（168 - 200 bpm）
  • Vivace － 活泼的（~140 bpm）
  • Allegro assai － 颇快的
  • Allegro － 快板（120 - 168 bpm）
  • Allegro Moderato － 适度、中速的快板
  • Allegretto － 小快板 （104 - 112 bpm）
  • Moderato － 中板（90 - 115 bpm）
  • Andantino － 小行板（比行板稍快）
  • Andante － 行板（76 - 108 bpm）
  • Adagietto － 小柔板 / 颇慢
  • Adagio － 柔板（66 - 76 bpm）
  • Grave － 庄严的慢板
  • Larghetto － 小慢板（60 - 66 bpm）
  • Lento － 慢板（40 - 60 bpm）
  • Largo － 最缓板或广板
  • Lentissimo － 甚缓
  • Larghissimo － 极端地缓慢（40 bpm 或以下）

修饰语

  • assai － 非常。例：Allegro assai 表示“甚急速地”
  • con brio － 有活力地
  • con moto － 有动态地
  • non troppo － 不过多地。例：Allegro non troppo （或 Allegro ma non troppo）表示“不太过分地急速”
  • non tanto － 不多地
  • molto － 非常。例：Molto Allegro 或 Adagio Molto 表示“极急速地”或“非常缓慢地”
  • poco － 稍微地。例：Poco Adagio 表示“略为缓慢地”
  • più － 更多地。例：Più Allegro更快，常见于拍子转变时相对的指示
  • meno － 较少地。例：Meno Presto
  • poco a poco － 逐步地

情感标记

  • Vivace 或Vivo－ 活泼的，一般指示快速的节奏
  • Maestoso － 雄伟的或庄重的，一般指示隆重的缓缓的节奏
  • Sostenuto － 持续的，拍子有时会有点放松减缓
  • Dolce － 亲切地、惬意地
  • Morendo － 垂死的

速度上的变动

  • Accelerando － 渐快（省略为 accel.）
  • Ritardando － 渐慢（省略为 rit. 或者 ritard.）
  • Meno Mosso － 缓慢点
  • Più Mosso － 稍快点
  • Rallentando － 慢下来，特别用于章节结束的地方（省略为 rall.）
  • Ritenuto － 临时地减慢（省略为 riten.）
  • Stretto － 临时地加快
  • Rubato － 弹性速度发挥
  • Allargando － 逐渐减慢转强，特别用于作品结束的地方
  • a tempo － 用于渐快和渐慢标记之后，表示回复渐快或渐慢之前的速度
  • Tempo primo 或 Tempo I － 如果乐曲有多次速度转变，Tempo primo 表示回复乐曲一开始的速度，例如一首三段曲式的作品，可以是Allegro - Andante - Tempo Primo

强弱

两个用来表示相对强弱程度的基本符号是：

  • p —意大利语 piano 的缩写，表示弱
  • f —意大利语 forte 的缩写，表示强

在这两个基本符号以上建立的另外一些符号包括：

  • mp —意大利语 mezzo-piano 的缩写，表示中等偏弱（中弱）
  • mf —意大利语 mezzo-forte 的缩写，表示中等偏强（中强）

叠用p和f，可以衍生出以下符号：

  • pp —意大利语 pianissimo 的缩写，表示非常弱
  • ff —意大利语 fortissimo 的缩写，表示非常强
  • ppp —意大利语 pianissimissimo 的缩写，表示极弱
  • fff —意大利语 fortissimissimo 的缩写，表示极强

常用的强弱符号中，强弱的次序如下：

ppp pp più p p mp mf f più f ff fff
弱 强

###　突变符号

  • sfz —意大利语 subito forzando 或 sforzando 的缩写，表示对这个音符突别加强
  • sfp —意大利语 subito forzando-piano 或 sforzando-piano 的缩写，表示在对这个音符突别加强后立即会到弱

渐变符号

  • crescendo（简作 cresc.）：渐强
  • decrescendo（简作 decresc.）：渐弱
  • diminuendo（简作 dim.）：渐弱

其他

  • al niente：渐弱至到无声
  • calando：渐弱
  • crescendo：渐强
  • decrescendo 或 diminuendo：渐弱
  • perdendo 或 perdendosi：渐弱至无声
  • morendo：渐弱至无声
  • marcato：突强
  • sotto voce：低声地
  • in rilievo：代表某一个乐器的音量比其他突出
